# Modern Personal Portfolio (Full-Stack)

A premium, responsive personal portfolio with a dedicated Node.js/Express backend.

## 🚀 Features

- **Premium UI**: Glassmorphism, smooth animations, and responsive layouts.
- **Full-Stack**: Dedicated backend for handling contact form submissions.
- **Interactivity**: Theme switching, smooth scrolling, and scroll-triggered animations.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (Vanilla).
- **Backend**: Node.js, Express.js.
- **Icons**: Lucide Icons.
- **Fonts**: Inter & Outfit (Google Fonts).

## 🏃 How to Run Locally

### 1. Prerequisites
- **Node.js** (v14 or higher) installed on your system. [Download here](https://nodejs.org/).

### 2. Automatic Setup (Windows)
Double-click the **`run.bat`** file in this folder. It will:
- Check for Node.js.
- Install backend dependencies.
- Start the backend server on `http://localhost:5000`.
- Open the frontend in your default browser.

### 3. Manual Setup

#### Backend:
1. Open a terminal in the `backend` folder.
2. Run `npm install` to install dependencies.
3. Run `npm start` to start the server.

#### Frontend:
1. Simply open `portfolio/index.html` in any web browser.

## 📂 Project Structure

- `/portfolio`: Frontend source code.
- `/backend`: Node.js/Express backend code.
- `run.bat`: Automation script for Windows.
